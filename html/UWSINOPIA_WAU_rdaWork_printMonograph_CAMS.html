Property Template: has expression of work(*)

For a collection or a parallel aggregate, use this template to describe an aggregating work. Relate the aggregating work to a manifestation using "has manifestation of work". For an augmentation aggregate, use this template to describe a work realized by the primary expression. Relate the work to the primary expression using "has expression of work".

  • Property IRI: http://rdaregistry.info/Elements/w/P10078
  • RDA Toolkit URL: https://access.rdatoolkit.org/en-US_ala-e23bf7e1-aa3e-3e8d-874e-9a16d0cdee57

  • Repeatable
  • Property type: lookup
  • Value constraint(s):

      • Sinopia RDA expression : urn:ld4p:sinopia:rda:expression
      • Sinopia RDA manifestation : urn:ld4p:sinopia:rda:manifestation
