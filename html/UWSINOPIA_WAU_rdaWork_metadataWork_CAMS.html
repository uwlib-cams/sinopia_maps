has identifier for work

Cataloging Guidance 

    An appellation of work, such as identifier for work, is required for minimum description of a work.

    Use the IRI for the metadata work to construct a value. Record the IRI portion following 'https://api.sinopia.io/resource/' as a literal value. (For test cataloging in Sinopia - Stage, use the portion following 'https://api.stage.sinopia.io/resource/'.)

has creator person of work
